Getting Started with Python

Posted by Ja'Crispy on February 8, 2022

In this tutorial you will learn the important things that youll need to have or know to start coding with Python. It is important that you have the required things and some base knowledge before you can start coding in Python. This tutorial will set you up with visual studio code but I left some other options linked at the bottom. You should explore the other options and maybe more options by googling.

Step One

I am going to show you guys how to install an IDE. An IDE is a Integrated Development Environment. It is where you write the code and in most cases, you can run the code inside the IDE as well. Without an IDE, its very hard to code. Itll tell you what your errors are, sometimes while writing code itll even show it. Itll also help make it easier to read and understand the code

Installing and Setting up VScode

In order to install visual studio code, you just have to go to their website.

Click Here Install Visual Studio Code

After installing VScode, open it

Youll need to install Python onto visual studio code, see screenshots below

After finding the Python extension, youll have to install it. This will allow you to make and run .py files, itll also give you a Python terminal. If you want to use VScode for Python than youll need to install the extension

Starting to Code

Now you can click the top button again to open the files. Than you can make a new file, and make sure it ends in .py than itll be a Python file. Than you can open the file and start coding from there.

Other Options

There is other options for IDE's but Visual Studio Code is one of the better ones because its used by a ton of people but it also has support for practically any programming language. But there is other options that you can go with if youd like to. I personally like VScode for a lot of reasons but there is no "Correct" option. All IDE's are good. Just look at some of the options below and decide from there what you want to use.

One of those other options is an IDE called PyCharm. It is a IDE for Python only and is another one of the top used IDE's for Python programming. Its used by a lot of people and it also has a ton of very nice features. Just like VScode it also has predictive auto complete. It also has a built in terminal as well. And it is also pretty customizable and can be used to code in other language.

Click here to go to PyCharm

There also online options, like Repl.it, Repl.it is an online IDE that you can use to code in a webpage. It can also link to github for easier use. It is nice for people at school who cant download anything to their computer, they can just go on the website and start coding.
