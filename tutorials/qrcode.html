QRcode Generator

Posted by Ja'Crispy on February 22, 2022

In this tutorial you'll learn how to make a qrcode generator for any URL you want. This can be useful in many situations but its also just a cool thing that you can learn how to do. We will make this script by using a module called qrcode. Although you might need to install a module called Images but its not always needed. If you get an error that says something like "No Module called Images" than just install that too. But what do we need to do to make this code come to life. We need to set it up to ask the user for the url and the name they want for the file. Than we need to use the qrcode module to make it and than save it.

Finisihing the code

Now all we need to do is actually create the qrcode, We will use the variable that we made earlier called url. Than we will save the qrcode as a jpg and the name will be "whatevernameyouinput.jpg". And thats it. Its super easy and simple.

code = qrcode.make(url)
code.save(f"{name}.jpg")

The full code

import qrcode

url = input("What is the URL? ")
name = input("What do you want to call the file? ")

code = qrcode.make(url)
code.save(f"{name}.jpg")
