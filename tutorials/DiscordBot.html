How to code a Discord Bot using Python

Posted by Ja'Crispy on February 3, 2022

This tutorial will teach you how to build a simple Discord bot using Python. We wont go over any advanced commands but we will start with a few very basic commands. In the future I plan on doing more advanced discord bot tutorials

Starting Step

To start with this tutorial youll need to go to the discord developers portal and create a bot

Click here to go to Discord Dev Portal

Creating the bot

In the top right of the developer page, you should see a new application button, click that and set a name for the bot

After you click that youll see a button that says add bot, click that. Next you should see a button that says copy, right under the token, youll need this token to make the bot actually do things

Installing the discord module

We can start the code now. Youll need to install the discord module and the time module

pip install discord
pip install time

Starting to code

The first thing youll want to do is import discord as well as setup a prefix. A prefix is the little thing before the command so the bot knows your talking to it

import discord
from discord.ext import commands
import time

client = commands.Bot(command_prefix="!")

After all the imports are done, than itll setup the prefix. For this tutorial, for the prefix I chose to use ! but you can use whatever youd like to

Setting bots presence 

Bots can have an activity status as well, they can also have a playing status. Most people use the playing status to display their prefix or any other information they would like to use

@client.event()
async def on_ready():
    await client.change_presence(status=discord.Status.idle, activity=discord.Activity(type= discord.ActivityType.playing, name=f"Prefix: !"))
    intents = discord.Intents(messages=True)


Now the bots activity status should be "prefix: !", and its status should also be idle. You can set these to be whatever you want them to be though. It also sets the bots intents for message sending to be true, this allows the bot to send messages

First command

Now that all of the setup is out of the way, we can get to the command part of the discord bot. We will just be making a simple command to send a message to the server in the channel that the message was sent in.

@client.command()
async def ping(ctx):
    await ctx.send("Pong!")

This will make the bot send "Pong!" back to the channel that the user sent the command in. This tells you that your bot is active and responding to commands.

Second command

This command will make the bot repeat whatever you added after the command and delete the original command, any user can run this command too

@client.command()
async def echo(ctx, *, echocommand):
    time.sleep(.5)
    await ctx.send(echocommand)

This command has an argument, the argument is called "echocommand", its kind of like a variable. So the bot will wait for half a second and than send back the argument, the argument is whatever the user types in after the command

Running the bot

Now how do we actually get the bot online and running? Well youll need to go back to the step where we got the bots token, get that again

print("Bot Online")
client.run("bots token here")

the 1st line will print to the console that the bot is online. To actually make the bot run you have to put your token where it says "bots token here", make sure you keep the "" though, otherwise it wont run

Thank you for reading
