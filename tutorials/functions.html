Basic Functions

Posted by Ja'Crispy on February 7, 2022

In this tutorial you will learn the basics of functions. Youll learn what they do, why there theyre, as well as how arguments work. Functions are an important part of Python. They are very useful and not too hard to get the idea of. I plan of making a series of smaller articles with Python basics.

What is a function

A function is like a class that you can call back to and run. The way I think is best to explain it, is as its own little sections of code. So you could have one section of code that you dont want to run until later or if its called on. So its a good way to make it so the code only runs if something happen. But it also helps with making it look more organized as well

How to Make a Function

def main():
    any code can go here

So how does that work? Well you start off by doing "def" this makes it so it defines the function that you want to make, next youll put in the title for example: def titlehere. Than you set the arguments that the function has which is none for this example so its left as (): so thats how you create a function.

Arguments

Arguments arent always needed but sometimes youll want to use them. An example of an argument is a variable. So say something was defined before the function but the code inside the function needs to use that variable. You can set the variable to be one of the arugments nad it can gather data from the variable

variable = 4
def main(variable):
    print(variable)
main(variable)

So the variable was defined outside of the function, but we want the function to be able to use the variable. So we set the variable to be an argument used by the function, that is why in the main() inside the () we had the variable name, that allows it to see or use the variable. than when we ran the function we had to add the argument in there too so where it says main(variable) its running it with the variable as an argument so it can use it.

Other Information

So that is the basics to functions, there is a little bit more you can use but almost every Python script should use functions. It makes things a little bit cleaner looking over all which makes for easier to read code. But it also allows for running functions if something happens but if something doesnt happen than the function doesnt run. Functions can be created without being ran too.
