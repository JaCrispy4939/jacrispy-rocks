How to code a Urban Dictionary Web Scraper using Python

Posted by Ja'Crispy on February 2, 2022

Building a web scraper in Python might seem like a hard task but it isnt the hardest thing to do

Starting Step

To start off youll need to install Beautiful soup and urllib3, youll also need to install the time module

pip install bs4
pip install urllib3
pip install time

Setting up the search

In order to start the search for the word, we need to write a code that has the URL and asks for the word that the user would like to search for


import urllib.request
from bs4 import BeautifulSoup
import time


def main():
    word = input("Whats word would you like defined: ")
    url = "https://www.urbandictionary.com/define.php?term="  + word + ""
    if word == "":   
        print("Please input a word")
        time.sleep(4)  
        main()

The 1st line sets up a function that can be run later on in the code
The 2nd line asks the user for whatever word they want to know the meaning of
and the second line defines the url to be urbandicionary and it also adds variable "word" to the end of it, meaning whatever the user types in for the word is what will be added to the end of the url 
and finally the 3rd line makes it so it runs the next 3 lines
The 4th line prints a warning to put a word
The 5th line makes the code wait for 4 seconds before continuing 
And the 6th line reruns the main function

Getting the Data

In order to gather the data off the website you have to go into Inspect Element, this can be done with (CTRL-SHIFT-I) or a right mouse click. Than you have to find the classes where the data is stored and copy the class name and paste it into the code

    htmlfile = urllib.request.urlopen(url)
    soup = BeautifulSoup(htmlfile, 'lxml')
    urban_definition = soup.find(class_="meaning").text
    urban_example = soup.find(class_= "example").text

The 1st line of code makes it so that the code can open the url that we set using the "url" variable
The 2nd line of code makes it so that beautiful soup is defined and has the link to open
The 3rd line of code finds the definition class and the .text part makes it so it only extracts the text
And the 4th line does the same as the 3rd but for the example

Finishing the code

The last thing you need to do is print the data that we found to the terminal so that the user can know the definition

    print(f"Meaning: {urban_definition}")
    print(f"Example: {urban_example}")
main()


The 1st line prints out the definition of the word, the little f makes it so you can add the variable into the line without needing a +
The 2nd line prints the example of the word that you wanted defined
And the 3rd line runs the code
