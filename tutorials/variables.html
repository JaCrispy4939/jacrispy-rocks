Variable Basics

Posted by Ja'Crispy on February 9, 2022

In this tutorial you will learn the basics of Variables. variables are a very important prt of coding. Without variables a lot of codes arent even able to be ran. Its very important to learn, understand and know how to use variables.

What is a Variable

A variable is a way to store some data. It can be called back to by the code and the data that it has written to it can be read, changed or used for an if type of statement. But it can also be set by the user when the code is ran. A variable is very important in most codes. For example, the Urban Dictionary webscraper tutorial we did earlier. That cant run without variables because it asks the user what word they want to use and it sets the variable to be what the user says. you can also have an unlimited amount of variables.

What it looks like

X = "JaCrispy"
number = 4
userchoice = input("What would you like: ")
list = ["Python", "Java", "C", "Javascript"]

Those are a few examples of variables and how you can put information or data into them. The general way it is done is anyvariablenamehere = whateverdatainanyway. You can just put in text data, like words, by doing varname = "" with your text inside the "". The next type I showed was a number type. That can be done by varname = 2, just set it equal to a number. And the third type I showed was getting user input. It will get input from the user and than itll set the variable to be whatever the user types in. It can be done by doing varname = input("anytexthere"). And the final way of doing it is a list. Itll make it so the variable has more than one little bit of data. It can be done by doing varname = ["thing1", "Thing2", "thing3"], You need the brackets otherwise it wont work. Youll also need to use the "" and , that are in it. The , act as a seperator so it knows that its something different and new not just the one piece of data.
